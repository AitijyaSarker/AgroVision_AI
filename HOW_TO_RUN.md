# 🚀 How to Run Agro Vision

## Quick Start (3 Steps)

### Step 1: Open Terminal
- In VS Code/Cursor: Press `` Ctrl+` `` (backtick key)
- Or: Right-click in file explorer → "Open in Terminal"

### Step 2: Navigate to Project Folder
```bash
cd "F:\Agro Vision"
```

### Step 3: Start the Server
```bash
npm run dev
```

**That's it!** 🎉

---

## What You'll See

```
▲ Next.js 14.2.35
- Local:        http://localhost:3000
✓ Ready in 7s
```

---

## Open in Browser

**Click the link** that appears in terminal, or manually go to:

```
http://localhost:3000
```

(If port 3000 is busy, it will use 3001, 3002, etc. - just use whatever port it shows!)

---

## Stop the Server

Press `Ctrl+C` in the terminal

---

## Restart the Server

1. Stop: `Ctrl+C`
2. Start: `npm run dev`

---

## Troubleshooting

### "Port is in use"
- Just use the port it shows (3001, 3002, etc.)
- Or kill the process using that port

### "Command not found"
- Make sure you're in the project folder: `cd "F:\Agro Vision"`
- Check Node.js is installed: `node --version`

### "Dependencies not found"
- Run: `npm install`
- Then: `npm run dev`
