# 📸 Farmer Image Setup

## How to Add the Farmer Image

1. **Save the farmer image** to the `public` folder
2. **Name it:** `farmer-rice-field.png` (or `.jpg`)
3. **The image will automatically appear** on the homepage!

## Image Requirements

- **Format:** PNG or JPG
- **Recommended size:** 600x600px or larger
- **Name:** `farmer-rice-field.png` (or `.jpg`)

## If Image Not Found

The website will automatically show a **cartoon-style fallback illustration** with:
- Animated rice field background
- Farmer emoji with phone
- Floating clouds and rice stalks
- All in cartoon style!
